Vocabulary Guide для UI/UX Дизайнера

Foundations

User Interface (UI)
Користувацький інтерфейс. Візуальна частина програми/сайту, з якою взаємодіє користувач.

"We need to simplify the user interface to reduce confusion." ➤ Потрібно спростити інтерфейс для зменшення плутанини.
User Experience (UX)
Досвід користувача. Загальне враження від продукту.

"Improving the UX helped us increase user retention." ➤ Поліпшення UX підвищило утримання користувачів.
Usability
Зручність у використанні. Наскільки легко досягти цілей.

"Improving the usability of the checkout page reduced cart abandonment by 30%." ➤ Поліпшення usability зменшило відмови на 30%.

Interface Components

Button
Кнопка.

"The 'Buy Now' button needs to stand out more." ➤ Кнопка «Купити зараз» має більше виділятися.
Input field
Поле введення.

"Please enter your email address in the input field." ➤ Введіть електронну адресу в поле введення.
Dropdown
Випадаючий список.

"Add a dropdown menu for language selection." ➤ Додати випадаючий список для вибору мови.
Checkbox
Чекбокс.

"Tick the checkbox to agree to the terms and conditions." ➤ Позначити чекбокс для погодження з умовами.
Toggle
Перемикач.

"Use the toggle to switch between dark and light mode." ➤ Перемикач режиму світла/темряви.

UX Design

User flow
Потік користувача.

"Map the user flow for the registration process." ➤ Побудуй потік користувача для процесу реєстрації.
Wireframe
Каркас/макет.

"Let's create a wireframe before moving to the high-fidelity design." ➤ Створимо каркас перед детальним дизайном.
Prototype
Прототип.

"This clickable prototype will help us test the navigation early." ➤ Клікабельний прототип допоможе ранньому тестуванню навігації.
Persona
Персона користувача.

"We designed this feature based on our user persona 'Anna'." ➤ Ми розробили цю функцію виходячи з персони Анни.
A/B Testing
A/B тестування.

"We ran an A/B test to choose the best button color." ➤ Ми провели A/B тест для вибору кращої кнопки.

UI Practices

Typography
Типографіка.

"Use consistent typography across all pages." ➤ Однакова типографіка на всіх сторінках.
Hierarchy
Ієрархія.

"We adjusted the hierarchy to highlight call-to-action." ➤ Ми скоригували ієрархію для підкреслення CTA.
White space
Порожній простір.

"Add more white space to improve readability." ➤ Додати більше простору для кращого сприйняття.
Contrast
Контраст.

"Increase contrast between text and background." ➤ Підвищити контраст між текстом і фоном.
Consistency
Послідовність.

"Ensure consistency across all navigation menus." ➤ Забезпечити послідовність навігації.

Process and Tools

Design system
Дизайн-система.

"We use a design system to ensure consistency." ➤ Використовуємо дизайн-систему для узгодженості.
Component
Компонент.

"Each button is a reusable component." ➤ Кожна кнопка — це компонент.
Responsive design
Адаптивний дизайн.

"Responsive design ensures usability across devices." ➤ Адаптивний дизайн забезпечує зручність на всіх пристроях.
Grid system
Сітка.

"Use the 12-column grid system for layout." ➤ Використання 12-колонної сітки для макетів.

Communication

Feedback
Зворотній зв'язок.

"The client gave positive feedback." ➤ Клієнт дав позитивний фідбек.
Deliverables
Робочі матеріали.

"All deliverables must be ready by Friday." ➤ Всі матеріали мають бути готові до п'ятниці.
Deadline
Кінцевий термін.

"We need to finalize before the deadline." ➤ Потрібно завершити до кінцевого терміну.
Revision
Перегляд, правка.

"The client asked for a revision of colors." ➤ Клієнт попросив переглянути кольори.
Handoff
Передача дизайну розробникам.

"Make sure components are labeled before handoff." ➤ Переконайтеся, що всі компоненти підписані перед передачею.
